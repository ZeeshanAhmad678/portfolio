To Do Now

A simple and clean task manager built using HTML, CSS, and JavaScript. Users can easily add new tasks, edit them through a prompt, and delete them with a click. Task data is saved in localStorage, ensuring it remains even after refreshing the page. Designed for simplicity, as a beginner.

Github Link: https://github.com/ZeeshanAhmad678/to-do-list

Silly Story Generator

The Silly Story Generator creates random, funny stories using user input and chance. It includes a custom name field and a U.S./U.K. unit conversion toggle for added flexibility. Each story is uniquely generated for a fun, unpredictable experience. Built with vanilla JavaScript, it's a light and creative project.

Github Links: https://github.com/ZeeshanAhmad678/silly-story

A simple and interactive Number Guessing Game built with HTML, CSS, and JavaScript. Players guess a random number with hints provided for too high or too low guesses. It's a beginner-friendly project that showcases basic logic and DOM manipulation skills.

Github Links: https://github.com/ZeeshanAhmad678/guess-number-game
